The NI-XNET VeriStand Custom Device enables you to use your NI-XNET hardware natively in NI-Veristand. 

For more information about the custom device, please see the developer zone document.

To use the NI-XNET VeriStand Custom Device, copy the folder named "NI-XNET" into the <Public Documents>\National Instruments\VeriStand\Custom Devices folder and restart NI VeriStand.

Add the NI-XNET Custom Device to a System Definition file by right clicking Custom Devices and selecting National Instruments >> Embedded Data Logger.

To view or edit the source code for the NI-XNET Custom Device, open the LabVIEW project in the Source folder.
